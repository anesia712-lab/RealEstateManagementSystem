# Skyline Realtor — Real Estate Management System

VTDI **Enterprise Application Development** coursework: Windows **WinForms** desktop (Module 1), shared **class library** (Module 2), **ASP.NET Core MVC** web app (Module 3), and **SQL Server**.

### For your lecturer (PDFs)

Hand-off documents live in **`uploads/`**: **GET_STARTED.pdf** (Windows and Mac), **EAD_SUBMISSION_CHECKLIST.pdf**, **DATABASE_CREDENTIALS.pdf**, **GROUP_MEMBERS.pdf**, plus **`RealEstateDB_Backup.bak`** and **`database_schema.sql`**. See **`uploads/README.md`**.

### For you (developers)

Open **`RealEstateManagementSystem.sln`**. Run the web project from **`RealEstateManagementSystem/`** with `dotnet run` after SQL is available. Submission layout and paths: **`EAD_UPLOAD_PACKAGE.md`**.
